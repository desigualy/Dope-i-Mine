
# Branding Assets

This directory holds the visual identity system for Dope-i-Mine.

Expected final exports:
- `dope_i_logo_lockup.*`
- `dope_i_mine_logo_mark.*`
- `dope_i_mine_wordmark.*`
- `mascot_brand_sheet.*`

Guidelines:
- keep the wordmark readable at small sizes
- Dope-i should feel friendly, not childish
- avoid loud medical/productivity clichés
- branding must work in both light and dark contexts

These files are briefs and placeholders until the final design pack is exported.
