# Mascot Expression Guide

Dope-i expressions must retain the neon hoodie robot mascot identity:

- black visor
- cyan glow facial features
- black hoodie
- headphones
- purple/cyan neon accents
- rounded 3D proportions

Do not reinterpret Dope-i as a block, Lego, or toy character.

Default expressions:
- idle
- listening
- thinking
- happy
- proud
- celebrating
- gentle concern
- speaking
