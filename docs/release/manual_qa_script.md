# Manual QA Script

## Auth
1. Create a new account
2. Log out
3. Log back in
4. Confirm redirect behavior

## Onboarding
1. Run through onboarding from scratch
2. Change assistant name
3. Change pronunciation
4. Save profile
5. Confirm home loads

## Tasks
1. Create a simple task
2. Confirm steps appear
3. Toggle minimum version
4. Break one step down further
5. Complete a step
6. Open overwhelm rescue sheet

## Routines
1. Create a routine
2. Open routine list
3. Start routine run
4. Complete step sequence

## Reminders
1. Save reminder
2. Verify persistence after app restart

## Voice
1. Play assistant name preview
2. Speak task aloud if STT is enabled on device

## Caregiver
1. Link caregiver
2. Assign routine
3. Verify assigned routine visibility

## Settings
1. Change companion/avatar
2. Change voice profile
3. Change pronunciation settings
