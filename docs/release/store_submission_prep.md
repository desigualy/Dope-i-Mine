# Store Submission Prep

## App Store / Play Store materials
- final app icon
- screenshots
- short description
- full description
- privacy policy URL
- support URL
- release notes

## Compliance review
- confirm no medical claims
- confirm accessibility is not premium-gated
- confirm voice usage is described clearly
- confirm caregiver features are consent-aware

## Submission advice
Keep the positioning practical:
'Break big tasks into tiny wins.'
Avoid overclaiming neuroscience or therapy outcomes.
